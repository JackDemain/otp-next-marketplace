# OTP NEXT — Витрина маркетплейса

> ⚠️ **Дисклеймер:** Это независимый концептуальный прототип. Проект **не является официальным продуктом ОТП Банка** и не аффилирован с АО «ОТП Банк» или OTP Group. Все товары, поставщики, цены и данные — вымышленные и используются исключительно в демонстрационных целях. Логотип и фирменный стиль ОТП Банка использованы для визуализации концепции.

B2B-маркетплейс фармацевтики и медицинских изделий с встроенными финансовыми сервисами.

## Что внутри

- 1,700+ товарных позиций, 30 поставщиков, 12 категорий
- Каталог с фильтрацией, поиском, сортировкой
- Рассрочка 6/12/18 месяцев на каждом товаре
- Модальное окно с болями рынка и решениями платформы
- Форма заявки на бета-тест

## Технологии

React + Babel (CDN), single-page, zero build tools.

## Запуск локально

Просто открой `index.html` в браузере. Или через сервер:

```bash
python3 -m http.server 8000
# → http://localhost:8000
```
